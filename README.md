# OXYDE
Lightweight library for testing and documenting RESTful APIs.

# Why
There are few approaches to test and document RESTful APIs.
Some of them are excellent in testing and some of them in documenting RESTful APIs.
We were looking for a tool that combines both of these capabilities.
We need a tool that allows us to automate building tests and documentation
keep the API in sync with documentation. This tool's name is **OXYDE**.

## With OXYDE you can
- write your tests extremely fast
- structure your tests as you wish
- test all RESTful API methods
- generate your API documentation 
- automate testing process
- keep your API in sync with documentation
- publish your documentation
